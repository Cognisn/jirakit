# Release Notes - v0.1.2

**Release Date:** 20 March 2025
**Type:** Minor Release

## Overview

Version 0.1.2 removes the dependency on the `mistune` markdown library, streamlining the codebase and reducing external dependencies.

## 🔧 Changes

### Removed mistune Dependency

- **Removed:** `mistune` markdown parsing library
- **Reason:** Simplified markdown handling
- **Impact:** Reduced dependency footprint

### Code Cleanup

- Removed unused markdown parsing code from `src/jirakit/issues/__init__.py`
- Streamlined text processing implementation

## 📦 Files Changed

**2 files changed, 2 insertions(+), 17 deletions(-)**

- `src/jirakit/_version.txt` - Version bump to 0.1.2
- `src/jirakit/issues/__init__.py` - Removed mistune usage

## ⚠️ Breaking Changes

None. The mistune library was not part of the public API.

## 🔄 Upgrade Instructions

1. Update to v0.1.2:
   ```bash
   pip install --upgrade jirakit
   ```

2. If you have mistune in your requirements, you can safely remove it (unless used elsewhere)
