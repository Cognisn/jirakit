# Release Notes - v0.1.6

**Release Date:** 9 October 2025
**Type:** Major Feature Release

## Overview

Version 0.1.6 is a significant release that introduces comprehensive documentation, deployment tracking with rollback capabilities, and a complete test suite. This release transforms jirakit from a basic library into a production-ready solution for Jira Cloud automation.

## ✨ New Features

### 📊 Deployment Tracking System

**DeploymentTracker** - Track all template deployments with precision rollback capabilities.

- **Automatic Resource Tracking:**
  - Tracks all created resources (issue types, screens, workflows, etc.)
  - Records deployment metadata (who, when, what template)
  - Captures status (in_progress, completed, failed, partial)
  - Logs errors with detailed context

- **Precision Rollback:**
  - **Tracking-based rollback:** Uses deployment tracking file for exact resource deletion
  - **Fallback rollback:** Searches by project key prefix when tracking file unavailable
  - Safe project deletion with `enableUndo` support
  - Comprehensive rollback summaries

- **JSON Persistence:**
  - Deployment files stored in `.jirakit_deployments/`
  - Load and review deployment history
  - Audit trail for compliance and troubleshooting

**Key Methods:**
- `DeploymentTracker()` - Create new tracker
- `DeploymentTracker.load()` - Load existing deployment
- `track_custom_field()`, `track_issue_type()`, `track_screen()`, etc.
- `mark_completed()`, `mark_failed()`, `mark_partial()`
- `get_summary()` - Get deployment summary

## 🔄 API Changes

### New Classes

```python
from jirakit.projects.tracking import DeploymentTracker

# Create tracker
tracker = DeploymentTracker(
    project_key="PROJ",
    template_name="My Template",
    deployed_by="user@example.com"
)

# Track resources
tracker.track_issue_type(issue_type_id, "Task")
tracker.track_screen(screen_id, "Task Screen")

# Mark completion
tracker.mark_completed()

# Load existing deployment
tracker = DeploymentTracker.load("PROJ")
summary = tracker.get_summary()
```

### Enhanced Methods

```python
# Rollback with tracking
summary = client.projects().rollback_template_deployment(
    project_key="PROJ",
    delete_project=True,
    enable_undo=False
)
```
